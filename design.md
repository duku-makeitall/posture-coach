# [UI/UX 디자인 가이드] 거북목 자세 코치 (My Friend Turtle)

본 문서는 학생들이 컴퓨터 앞에서 친근하게 대화하듯 교류하며 바른 자세를 학습할 수 있도록 유도하는 **'거북목 자세 코치'** 웹사이트의 UI/UX 디자인 가이드라인입니다. 차가운 전문 진단 장비의 감성을 배제하고, 귀여운 캐릭터와 파스텔 네온 컬러를 활용한 감성 피드백을 구축합니다.

---

## 1. 디자인 컨셉 (Design Concept)

### "My Friend Turtle" (거북목 자세 코치)
*   **따뜻하고 포근한 어둠 (Warm Dark Navy)**: 눈의 피로를 예방하면서도 편안하고 친근한 학업/작업 공간을 위해 **따뜻한 딥 네이비(Deep Navy)** 배경 테마를 기본으로 합니다.
*   **동글동글한 모서리 (Round & Soft)**: 모든 UI 카드의 모서리 반경을 `24px` 이상의 둥근 형태로 디자인하여 날카롭고 위협적인 요소를 배제합니다.
*   **감성 캐릭터 인터랙션 (Emotion Feedback)**: 현재 자세에 따라 실시간으로 얼굴 표정과 모션이 변하는 **아기 거북이 캐릭터**를 화면 전면에 노출하여, 학생들이 억지로 자세를 고치는 것이 아니라 거북이 친구를 위해 스스로 자세를 바로잡고 싶게끔 행동 유도를 이끌어냅니다.

## 2. 컬러 시스템 (Color System)

차가운 하이테크 네온 컬러 대신 생동감 있고 부드러운 **소프트 파스텔 톤**의 컬러를 적용합니다.

```css
:root {
  /* 테마 배경색 */
  --bg-main: #0B0F19;               /* 포근하고 깊은 다크 네이비 */
  --bg-surface: rgba(30, 41, 59, 0.65); /* 둥글둥글한 반투명 유리 카드 */
  
  /* 텍스트 컬러 (친근한 폰트용) */
  --text-primary: #F3F4F6;          /* 맑은 우유빛 회색 */
  --text-secondary: #94A3B8;        /* 소프트 블루빛 회색 */
  --text-muted: #64748B;            /* 차분한 회색 */

  /* 거북이 상태별 테마 컬러 */
  --color-mint: #34D399;            /* 바른 자세 (초록 - "거북이 신남! 🟢") */
  --color-peach: #F87171;           /* 거북목 감지 (빨강 - "거북이 힘듦! 🔴") */
  --color-sky: #60A5FA;             /* 분석 중 (파랑 - "거북이 탐색 중... 🔵") */
  --color-yellow: #FBBF24;          /* 보정 대기 (노랑 - "기다리는 중! 🟡") */
  --color-gray: #475569;            /* 시리얼 연결 끊김 / 인식 실패 */

  /* 부드러운 파스텔 네온 글로우 */
  --border-glass: rgba(255, 255, 255, 0.08);
  --shadow-neon-mint: 0 0 20px rgba(52, 211, 153, 0.35);
  --shadow-neon-peach: 0 0 20px rgba(248, 113, 113, 0.35);
  --shadow-neon-sky: 0 0 20px rgba(96, 165, 250, 0.35);
}
```

## 3. 타이포그래피 (Typography)

학생들이 읽을 때 지루하지 않고 친근감을 느낄 수 있는 캐주얼 라운드 서체를 지정합니다.

*   **웹 폰트 Import 선언**:
    ```css
    @import url('https://fonts.googleapis.com/css2?family=Jua&family=Nunito:wght@600;800;900&display=swap');
    ```
*   **서체 구성 가이드**:
    *   **대제목, 수치, 상태 텍스트**: `family: 'Jua', sans-serif;` (동글동글하고 꽉 찬 한글 서체)
    *   **본문 설명글, 영문 수치**: `family: 'Nunito', sans-serif;` (끝이 둥근 모던 영문 서체)

| 분류 | 크기 (px/rem) | Weight (굵기) | Line Height | 용도 |
| :--- | :--- | :--- | :--- | :--- |
| **Title H1** | `32px / 2.0rem` | Regular (Jua 기본) | 1.3 | "거북목 자세 코치" 타이틀 |
| **Subtitle H2** | `20px / 1.25rem` | Regular (Jua 기본) | 1.4 | 카드 패널 제목 |
| **Angle Display**| `52px / 3.25rem` | Bold (900) | 1.0 | 실시간 목 기울기 수치 (각도) |
| **Body Regular** | `15px / 0.938rem`| Medium (500) | 1.5 | 버튼 설명, 사용자 가이드 텍스트 |
| **Friendly Alert**| `18px / 1.125rem`| Regular (Jua 기본) | 1.4 | 거북이 캐릭터 대사창 텍스트 |

## 5. 카메라 및 뼈대 드로잉 (Camera & Skeleton overlay)

기계적인 인체 탐지기 느낌을 중화하기 위해, 스켈레톤의 디자인 톤을 부드럽게 완화합니다.

*   **스켈레톤 라인 스타일**:
    *   기존의 강하고 형광색 선 대신 부드러운 라벤더 화이트 `rgba(230, 230, 250, 0.45)` 컬러를 사용하여 눈의 자극을 낮춥니다.
    *   두께는 `3px`로 둥글게 이어줍니다.
*   **측정 랜드마크 포인트**:
    *   귀(Ear)와 어깨(Shoulder) 위치에 **귀여운 둥근 도넛 형태의 포인트**를 렌더링합니다.
    *   바른 자세 상태일 때: 포인트 내부를 산뜻한 파스텔 민트색(`#34D399`)으로 채웁니다.
    *   거북목 굽힘 감지 시: 0.5초 동안 서서히 부드러운 피치 핑크색(`#F87171`)으로 변하게 만듭니다.

## 6. 캐릭터 연동 및 상태 컴포넌트 (Character & Status)

대시보드 우측의 메인 카드에는 현재 판정 상태에 매칭되어 작동하는 거북이 캐릭터와 대사가 표시됩니다.

| 상태 | 거북이 캐릭터 비주얼 피드백 | 웹 UI 대사창 텍스트 | 아두이노 명령 |
| :--- | :--- | :--- | :--- |
| **보정 대기** | 돋보기를 든 아기 거북이 🔍 | "바른 자세를 취한 뒤, 아래 '기준 자세 등록' 버튼을 눌러주세요." | `NEO:0\n` |
| **분석 중** | 탐정 모자를 쓴 아기 거북이 ⏳ | "측면 모습이 카메라에 잘 보이도록 서 주세요. 분석 중입니다..." | `NEO:3\n` |
| **바른 자세** | 미니 왕관을 쓰고 미소 짓는 거북이 👑 | "지금 자세가 아주 좋아요! 이 상태를 계속 유지해 볼까요?" | `NEO:2\n` |
| **거북목 자세** | 땀을 뻘뻘 흘리는 지친 거북이 💦 | "목이 조금 기울어졌어요. 어깨를 펴고 고개를 살짝 들어주세요!" | `NEO:1\n` |
| **인식 실패** | 물음표 판넬을 들고 갸우뚱하는 거북이 ❓ | "카메라 화면에서 보이지 않아요. 카메라 정면 안쪽으로 들어와 주세요." | `NEO:0\n` |

## 7. 마이크로 애니메이션 (Micro-interactions)

### 7.1 거북이 캐릭터 둥둥 애니메이션 (Floating Motion)
화면 속 거북이가 뻣뻣하게 멈춰 있지 않고 살아 움직이는 느낌을 주도록 부드러운 바운스 애니메이션을 영구 적용합니다.

```css
.friendly-turtle-character {
  animation: float-around 3s ease-in-out infinite;
  will-change: transform;
}

@keyframes float-around {
  0% { transform: translateY(0px) rotate(0deg); }
  50% { transform: translateY(-10px) rotate(2deg); }
  100% { transform: translateY(0px) rotate(0deg); }
}
```

### 7.2 대사창 말풍선 확대 트랜지션 (Bubble Scale)
거북이의 상태(대사)가 변할 때 말풍선이 튀어나오는 듯한 스케일 애니메이션 효과를 주어 이목을 끕니다.

```css
.dialog-bubble {
  transform-origin: bottom center;
  animation: pop-bubble 0.4s cubic-bezier(0.175, 0.885, 0.32, 1.275) forwards;
}

@keyframes pop-bubble {
  0% { transform: scale(0.8); opacity: 0; }
  100% { transform: scale(1); opacity: 1; }
}
```
